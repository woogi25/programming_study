다차원 배열 : 배열 항목에 또 다른 배열이 대입된 것입니다.

값 목록으로 다차원 배열 생성
타입[][] 변수 = {
{값1, 값2, ... },
{값3, 값4, ... },
...
};

new 연산자로 다차원 배열 생성
타입[][] 변수 = new 타입[1차원수][2차원수];

초기값을 가지는데 정수 타입은0, 실수 타입은 0.0, 논리 타입은 false, 참조 타입은 null 입니다.

int[][] scores = new int[2][];
scores[0] = new int[3];	//첫 번째 반의 학생 수가 3명
scores[1] = new int[2];	//두 번째 반의 학생 수가 2명
