자바의 문자열은 String 객체로 생성됩니다.
String name1 = "홍길동";
String name2 = "홍길동";
동일한 String 객체의 번지가 생성됩니다.

new 연산자는 새로운 객체를 만드는 연산자로 객체 생성 연산자라고 합니다.

String  name1 = new String("홍길동");
String  name2 = new String("홍길동");

이 경우 name1과 name2는 서로 다른 String 객체의 번지를 가지게 됩니다.

내부 문자열만을 비교할 경우에는 equals() 메소드를 사용합니다.

String 변수에 빈 문자열("")을 대입할 수도 있습니다.

문자 추출
charAt() 메소드 : 문자열에서 특정 위치의 문자를 얻을 때 사용합니다.	//0부터 시작합니다.

문자열 길이
length() 메소드 : 문자열에서 문자의 개수를 얻을 때 사용합니다. 공백을 포함합니다.

문자열 대체
replace() 메소드 : 문자열에서 특정 문자열을 다른 문자열로 대체하고 싶을 때 사용합니다.
replace(기존 문자열, 대체 문자열);	//완전히 새로운 문자열 객체입니다.

문자열 잘라내기
substring() 메소드 : 문자열에서 특정 위치의 문자열만 잘라내서 가져오고 싶을 때 사용합니다.
substring(시작 인덱스) : 시작 인덱스에서 끝까지 잘라내기
substring(시작 인덱스, 끝 인덱스) : 시작 인덱스에서 끝 인덱스 까지 잘라내기

문자열 찾기
indexOf() 메소드 : 문자열에서 특정 문자열의 위치를 찾을 때 사용합니다.
만약 주어진 문자열이 포함되어 있지 않으면 -1을 리턴합니다.

contains() 메소드 : 주어진 문자열이 단순히 포함되어 있는지만 조사하고 싶을 때 사용합니다.

문자열 분리
split() 메소드 : 구분자를 사용하여 여러개의 문자열을 분리하고 싶을 때 사용합니다.
