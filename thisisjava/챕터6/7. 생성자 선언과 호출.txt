클래스 변수 = new 클래스();

모든 클래스는 생성자가 존재하며, 하나 이상을 가질 수 있습니다.

기본 생성자
바이트 코드 파일(Car.class)

public class Car {
	public Car() {	//자동 추가
}
명시적으로 선언한 생성자가 있다면 컴파일러는 기본 생성자를 추가하지 않습니다.


생성자 선언
클래스(매개변수, ...){	//생성자 블록
	//객체의 초기화 코드
}

매개 변수의 타입은 매개값의 종류에 맞게 작성하면 됩니다.


생성자 오버로딩
매개변수를 달리하는 생성자를 여러개 선언하는 것을 말합니다.


다른 생성자 호출
공통 코드를 한 생성자에만 집중적으로 작성하고, 나머지 생성자는 this를 사용해서 개선합니다.
