메소드 선언 : 객체의 동작을 실행 블록으로 작성하는 것을 말합니다.
메소드 호출 : 실행 블록을 실제로 실행하는 것을 말합니다.

메소드 선언
리턴타입 메소드명 (매개변수, ...) {

	실행 코드 작성
}

리턴 타입
결과값의 타입. 리턴값이 없는 메소드는 void로 작성.
리턴 타입이 있는 메소드는 return 문으로 리턴값을 반드시 지정해야 합니다.

메소드명
첫 문자를 소문자로 시작하고, 캐멀 스타일로 작성합니다.
void run()
void setSpeed(int speed)
String getName()

매개 변수
double divide(int x, int y)
매개 변수 2개 필요

실행 블록
메소드 호출 시 실행되는 부분입니다.


메소드 호출
메소드 블록을 실행하는 것을 말합니다.


가변길이 매개변수
int sum(int ... values) {
}

메소드 호출
int result = sum(1, 2, 3);
int result = sum(1, 2, 3, 4, 5);


return 문
메소드의 실행을 강제 종료하고 호출한 곳으로 돌아간다는 의미입니다.
return [리턴값];


메소드 오버로딩
메소드 이름은 같되 매개변수의 타입, 개수, 순서가 다른 메소드를 여러 개 선언하는 것을 말합니다.
