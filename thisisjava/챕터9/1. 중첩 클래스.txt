중첩 클래스
클래스 내부에 선언한 클래스를 말합니다.

1. 멤버 클래스 : 클래스의 멤버로써 선언되는 중첩 클래스입니다.
1.1) 인스턴스 멤버 클래스
class A {
	* class B {...} *	//	1.2) 선언 위치
}

1.3) 객체 생성 조건
A 객체를 생성해야만 B 객체를 생성할 수 있습니다.

2.1) 정적 멤버 클래스
class A {
	* static class B {...} * //	2.2) 선언 위치	
}

2.3) 객체 생성 조건
A 객체를 생성하지 않아도 B 객체를 생성할 수 있습니다.

2. 로컬 클래스 : 메소드 내부에서 선언되는 중첩 클래스입니다.
class A {
	void method() {
		* class B {...} * // 선언위치
	}
}

객체 생성 조건
method가 실행할 때만 B 객체를 생성할 수 있습니다.

중첩 클래스도 하나의 클래스이기 때문에 컴파일하면 바이트코드 파일이(.class)이 별도로 생성됩니다.

멤버 클래스일 경우 바이트 코드 파일 이름
A		$	B		.class
바깥 클래스		멤버 클래스

로컬 클래스일 경우 바이트 코드 파일 이름
A		$1	B		.class
바깥 클래스		로컬 클래스
