네트워크는 여러 컴퓨터들을 통신 회선으로 연결한 것을 말합니다.
LAN은 가정, 회사, 건물 등 특정 영역에 존재하는 컴퓨터를 연결한 것이고,
이것을 연결한 것이 WAN입니다.
WAN이 우리가 말하는 인터넷입니다.

서버와 클라이언트
서버 : 서비스를 제공하는 프로그램입니다.
클라이언트 : 서비스를 요청하는 프로그램입니다.

먼저 클라이언트가 요청하고, 서버는 결과를 응답해줍니다.


IP 주소
컴퓨터의 고유한 주소입니다.
네트워크 어댑터(LAN 카드)마다 할당되며, 만약 컴퓨터에 2개의 어댑터가 장착되어 있으면
2개의 IP 주소를 할당받습니다.


연결할 상대방 컴퓨터의 IP 주소를 모르면 프로그램들은 서로 통신할 수 없습니다.
프로그램은 DNS를 이용해서 컴퓨터의 IP주소를 검색합니다.

DNS는 도메인 이름이며, IP를 등록하는 저장소입니다.


Port 번호
운영체제가 관리하는 서버 프로그램의 연결 번호입니다.
서버는 시작할 때 특정 Port 번호에 바인딩합니다.
클라이언트가 웹 서버와 통신하려면 해당 번호로 요청을 해야합니다.


클라이언트도 서버에서 보낸 정보를 받기 위해서는 Port 번호가 필요한데, 서버와
같이 고정적인 Port 번호가 아니라 운영체제가 자동으로 부여하는 번호를 사용합니다.
이 번호는 클라이언트가 서버로 요청할 때 함께 전송되고, 서버가 클라이언트로
데이터를 보낼 때 사용됩니다.
